Sistema de Detección Inteligente de Cascos

Basado en IA para aumentar la seguridad en entornos laborales.

Sobre el proyecto

Helmet AI utiliza visión por computadora y modelos YOLO para detectar el uso adecuado de cascos de seguridad en tiempo real.

Prueba el modelo

Sube una imagen y la IA detectará si se está usando casco.

Equipo

Integración de múltiples IAs, desarrollo robusto y eficiente.
